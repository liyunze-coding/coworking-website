## Pomodoro timer

A pomodoro timer is a great way to keep track of your work sessions. It’s also a great way to let your viewers know when you’re taking a break. You can use a pomodoro timer widget to display the current work session and break time.

### Browser Sources

#### [MohFocus' Minimal Pomo Timer 🔗](https://github.com/mohamed-tayeh/Minimal-Pomo-Timer)

<a href="https://github.com/mohamed-tayeh/Minimal-Pomo-Timer" target="_blank">
    <Image src={MohMinimalTimer} alt="Minimal Pomo Timer" loading="eager"/>
</a>

- Twitch only
- Easy to setup and customize 
- Called through commands in Twitch chat
- Optionally post break time notifications to Discord

#### [Pomodoro timer by Nicholas Allam 🔗](https://njallam.github.io/stream-pomodoro/?pomodoro=50&shortBreak=10&longBreak=5)

- Fast and easy to setup
- Interactive through OBS

#### [Mayuko's Pomodoro Countdown Timer 🔗](https://github.com/hellomayuko/Pomodoro-Countdown)

<a href="https://github.com/hellomayuko/Pomodoro-Countdown" target="_blank">
  <Image src={MayukoPomo} alt="Mayuko Pomodoro Countdown Timer" loading="lazy" />
</a>

- Easy to setup and customize
- Interactive through OBS

### Window Capture

#### [Pomofocus 🔗](https://pomofocus.io/)

<a href="https://pomofocus.io/" target="_blank"><Image src={Pomofocus} alt="pomofocus" loading="lazy" /></a>

- You can also add tasks to do on the web page

## Task widgets

A task list widget is a great way to track your tasks and your chatters!

<div class="flex justify-start items-end h-fit gap-10">
    <a href="https://www.focumon.com/for_streamers" target="_blank">
        <h2>Focumon's co-working widget 🔗</h2>
    </a>
    <div class="bg-[#319c2a] px-2 py-1 w-fit rounded-lg mb-3">Sponsored</div>
</div>

- Free with optional subscription
- Easy to use, set up, and customize
- Affiliate program to earn money
- Handles one task per user

### [Rython Task Bot v2 🔗](https://github.com/liyunze-coding/rython-task-bot-v2)

<a href="https://github.com/liyunze-coding/rython-task-bot-v2" target="_blank">
<Image src={MultiTask} alt="Multi Task Widget" loading="lazy" />
</a>

- Free and open source
- Requires Streamer.Bot to be setup
- Easy to use, setup and customize

### [Super Sweet Bot 🔗](https://www.patreon.com/super_sweet_bot)

<a href="https://www.patreon.com/super_sweet_bot" target="_blank">
<Image src={SuperSweetBot} alt="Super Sweet Bot" loading="lazy" />
</a>

- €3 / month
- Comes with a Pomodoro Timer
- Colors are customizable
- Hosted on a cloud server
- Users can use !backlog to retrieve their tasks from other streams

## Music Widgets

Display what you're playing on Spotify or YouTube Music! 

### [6K Labs 🔗](https://6klabs.com/)

<a href="https://6klabs.com/" target="_blank">
<Image src={SixKLabs} alt="6K Labs widget" loading="lazy" />
</a>

- Easy to implement
- Available for YouTube Music and Spotify
- Has many design choices and settings:
  - Dark/Light mode
  - Tint colour options
  - Different aesthetic choices:
    - Compact
    - macOS
    - Linux Shell
    - And many more!

### [Pixel Chat 🔗](https://pixelchat.tv/)

<a href="https://pixelchat.tv/" target="_blank">
<Image src={PixelChat} alt="Pixel Chat widget" loading="lazy" />
</a>

- Shows what you're playing via Spotify, Apple Music etc.
- Includes other widgets for shoutouts, captions and counters
